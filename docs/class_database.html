class Database

Esta clase contiene la informacion necesaria para la conexion a base de datos.

Esta alojada en el archivo :

CoockAPI/coock.database.php

Atributos

  • host
  • user
  • password
  • database

host

Es el nombre del servidor en el que esta alojada la base de datos.

user

Es el nombre de usuario para conectarse al gestor base de datos.

password

Es la clave de usuario para conectarse al gestor base de datos.

database

Es la base de datos a la que se hara la peticion de datos.

Metodos

  • Database
  • connect
  • query($sql)
  • insert($sql)
  • insert_id($sql)

Database

Es el constructor de la clase de conexion, al ser llamado inicializa las variables de conexion para tener acceso al servidor de base de datos para hacer las futuras peticiones.

connect > $con

Este metodo hace una conexion arbitraria a la base de datos y devuelve el identificador de conexion para futuras peticiones.

query($sql) > $query

Realiza una peticion $sql select directa al servidor de base de datos y si la peticion sale bien, la funcion devuelve un descriptor para hacer el recorrimiento de los resultados de la sentencia $sql select.

insert($sql)

Realiza una sentencia $sql insert directa al servidor de base de datos, igual que la anterior la funcion connect esta inerita.

insert_id($sql)

Realiza una sentencia $sql insert directa al servidor de base de datos, igual que la anterior la funcion connect esta inerita.Como resultado devuelve el identificador que se ha insertado en la base de datos.

Ejemplos

Conexion Basica

	include("CoockAPI/coock.database.php");
	$db = new Database();
	$con = $db->connect(); // $con, contendra el contexto de la conexion a la base de datos

Recorrido Basico

	include("CoockAPI/coock.database.php");
	$db = new Database();
	$q = $db->query("select * from users");
	while($r = $q->fetch_array())
	    print $r['username'];

insercion Basica

	include("CoockAPI/coock.database.php");
	$db = new Database();
	$id = $db->insert("insert into user (username,password) value ('usuariox','passwordx')");
	print $id;
